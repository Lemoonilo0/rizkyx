Materi DNS dan DNS Server

DNS (Domain Name System) adalah sistem yang digunakan untuk menerjemahkan nama domain seperti www.google.com menjadi alamat IP seperti 172.217.0.46 yang dapat dikenali oleh komputer. DNS berfungsi sebagai "buku telepon" untuk internet.

Fungsi DNS:

  • Mempermudah pengguna mengingat alamat website.
  • Mempercepat proses akses website dengan caching.
  • Menyediakan layanan redundansi melalui server DNS yang tersebar.

DNS Server: DNS server adalah server yang bertanggung jawab untuk menyimpan dan memberikan informasi tentang nama domain dan alamat IP. Contoh jenis DNS server:

  • Recursive Resolver: Menerima permintaan dari klien dan mencari jawaban dari DNS lain jika tidak ada di cache.
  • Root Server: Server utama yang menyimpan informasi tentang server otoritatif untuk setiap domain tingkat atas (TLD).
  • Authoritative Server: Menyediakan jawaban definitif untuk nama domain tertentu.
